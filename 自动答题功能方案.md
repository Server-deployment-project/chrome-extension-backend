# 自动答题功能方案说明

## 一、功能概述

### 1.1 核心能力

用户打开题目页面（如学习通、优学院），点击扩展的"自动答题"按钮，选择题目区域后，扩展自动识别题目并填写答案到页面表单中。

### 1.2 支持的表单类型

| 表单类型 | HTML元素 | 填写方式 |
|---------|---------|---------|
| 文本输入 | `<input type="text">` | 设置 `value` 属性并触发 `input` 事件 |
| 文本域 | `<textarea>` | 设置 `value` 属性并触发 `input` 事件 |
| 单选 | `<input type="radio">` | 设置 `checked = true` |
| 多选 | `<input type="checkbox">` | 设置 `checked = true` |

## 三、核心实现逻辑

### 3.1 题目与表单元素捕获

用户选择题目区域时，需要同时捕获：
1. **题目区域的截图**
2. **该区域附近的所有表单元素**

```typescript
// content.ts - 捕获逻辑（伪代码）
function captureQuestionWithForms(rect: SelectionRect): CaptureResult {
    // 1. 找到选择区域内的表单元素
    const formElements: FormElementInfo[] = []
    const selectors = ['input', 'textarea', 'select']

    selectors.forEach(selector => {
        document.querySelectorAll(selector).forEach(el => {
            const elemRect = el.getBoundingClientRect()
            // 检查元素是否在选择区域内，或与选择区域有交集
            if (isIntersecting(elemRect, rect) || isNearby(elemRect, rect)) {
                formElements.push({
                    tag: el.tagName.toLowerCase(),
                    type: (el as HTMLInputElement).type,
                    id: el.id,
                    name: el.getAttribute('name'),
                    value: (el as HTMLInputElement).value,
                    rect: elemRect,
                    // 用于后续匹配答案
                    index: formElements.length
                })
            }
        })
    })

    // 2. 返回捕获结果
    return {
        image: captureArea(rect),
        formElements: formElements,
        selectionRect: rect
    }
}

function isIntersecting(a: DOMRect, b: DOMRect): boolean {
    return !(a.right < b.left || a.left > b.right ||
             a.bottom < b.top || a.top > b.bottom)
}

function isNearby(a: DOMRect, b: DOMRect, threshold = 50): boolean {
    // 检查元素是否在选择区域附近（上下30px以内）
    return Math.abs(a.top - b.bottom) < threshold ||
           Math.abs(a.bottom - b.top) < threshold
}
```

### 3.2 答案匹配策略

由于不同网站的表单结构差异大，采用**基于位置的智能匹配**：

```typescript
// content.ts - 答案填写逻辑（伪代码）
function fillForms(answer: AutoAnswer, formElements: FormElementInfo[]) {
    // answer 结构：
    // {
    //   answers: [
    //     { index: 0, value: "正确答案" },
    //     { index: 1, value: "B" }
    //   ]
    // }

    answer.answers.forEach(ans => {
        const elem = formElements[ans.index]
        if (!elem) return

        const el = document.querySelector(`[name="${elem.name}"], #${elem.id}`)
        if (!el) return

        const input = el as HTMLInputElement | HTMLTextAreaElement

        switch (input.tagName.toLowerCase()) {
            case 'input':
                if (input.type === 'radio' || input.type === 'checkbox') {
                    input.checked = true
                    input.dispatchEvent(new Event('change', { bubbles: true }))
                } else {
                    input.value = ans.value
                    input.dispatchEvent(new Event('input', { bubbles: true }))
                    input.dispatchEvent(new Event('change', { bubbles: true }))
                }
                break

            case 'textarea':
                input.value = ans.value
                input.dispatchEvent(new Event('input', { bubbles: true }))
                input.dispatchEvent(new Event('change', { bubbles: true }))
                break
        }
    })
}
```

### 3.3 后端Prompt增强

为了让AI返回结构化答案，需要在Prompt中明确指定输出格式：

```
你是一个题目解答助手。请分析图片中的题目，并给出答案。

请严格按照以下JSON格式返回答案：
{
    "answers": [
        {"index": 0, "value": "第一个题目的答案"},
        {"index": 1, "value": "B"},
        {"index": 2, "value": true}
    ]
}

其中：
- index: 题目的索引位置（从0开始，与前端捕获的表单元素顺序对应）
- value: 答案内容（文本选择题目标识如"A/B/C/D"，判断题用true/false）
```

## 七、潜在问题与解决

### 7.1 表单匹配失败

| 场景 | 解决方案 |
|------|---------|
| 网站使用动态ID | 使用 `name` 属性匹配，或基于位置匹配 |
| 表单元素在题目区域外 | 扩大捕获范围，识别题目下方的表单 |
| 多个相似表单 | 让用户确认具体填写哪个 |

### 7.2 答案格式识别

| 场景 | 解决方案 |
|------|---------|
| AI返回文本答案但表单是单选 | 解析文本含义（如"答案是B" → 找到value="B"的选项） |
| 答案顺序与表单不一致 | 依赖前端传入的 `formElements` 的 `index` 字段 |

### 7.3 动态渲染页面

| 场景 | 解决方案 |
|------|---------|
| 表单通过JS动态加载 | 延迟捕获，等待DOM稳定 |
| 题库网站（iframe） | 需要 `iframe` 权限 |
